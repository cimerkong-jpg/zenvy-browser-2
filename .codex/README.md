# .codex

Bo huong dan lam viec cho Codex/coding agent trong repo nay.

## Cach Dung
Doc `CODEX.md` truoc. Sau do dung them:

- `rules/` khi can quy tac theo chu de.
- `roles/` khi task can mot goc nhin chuyen mon.
- `workflows/` khi can quy trinh thuc thi cu the.
- `skills/` khi can mot nang luc reusable ket hop role, rule, workflow, checklist.
- `checklists/` truoc khi ket luan hoac release.

## Skills Hien Co
- `codebase-orientation.md`: hieu project/module/code path.
- `feature-implementation.md`: them hoac sua behavior user-facing.
- `bugfix-debugging.md`: dieu tra va sua loi.
- `code-review.md`: review diff/code voi risk focus.
- `test-verification.md`: chon va chay check phu hop.
- `ui-ux-polish.md`: UI, UX, responsive, accessibility.
- `secure-config.md`: auth, secret, config, logs, permission.
- `dependency-upgrade.md`: dependency, lockfile, upgrade.
- `release-readiness.md`: check truoc khi ship/handoff.

## Bo Rule Nang Cao
- `rules/context.md`: context strategy cho AI agent.
- `rules/git.md`: lam viec an toan voi worktree.
- `rules/dependencies.md`: quan ly dependency va lockfile.
- `rules/error-handling.md`: error/logging policy.
- `rules/data-safety.md`: bao ve du lieu va migration.
- `workflows/decision-record.md`: format quyet dinh ky thuat.

## Nguyen Tac
- Day la bo instruction chung, khong gan rieng vao mot san pham.
- Khong luu secret.
- Khong thay the README cua project.
- Khong dung de ghi lich su task ngan han.
