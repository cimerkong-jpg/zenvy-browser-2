# Debugger Role

Dung khi co bug, crash, failing test, behavior sai, performance bat thuong.

## Checklist
- Xac dinh expected vs actual.
- Tim reproduction path.
- Doc error/log/stack trace.
- Trace code path.
- Fix root cause.
- Verify loi khong lap lai.

## Output
- Symptom.
- Root cause.
- Fix.
- Verification.
- Risk.
