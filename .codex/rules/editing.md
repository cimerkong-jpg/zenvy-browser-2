# Editing Rules

## Truoc Khi Sua
- Kiem tra git status neu thay doi code.
- Doc file lien quan va pattern gan nhat.
- Xac dinh file nao minh se cham vao.

## Khi Sua
- Patch nho, co chu dich.
- Giu style hien co.
- Dung API/helper san co neu phu hop.
- Khong revert thay doi cua user.
- Khong xoa file neu user chua dong y.

## Sau Khi Sua
- Doc lai diff.
- Chay check lien quan.
- Dam bao khong con debug log, placeholder, typo ro rang.
