# Frontend Rules

## UX
- Uu tien workflow that cua user.
- Co trang thai loading, empty, disabled, error, success khi phu hop.
- Khong de text overflow, overlap, layout shift.
- Giu UI scan duoc, khong trang tri qua muc voi tool van hanh.

## Accessibility
- Dung semantic controls khi co the.
- Co focus state.
- Khong dua vao mau sac duy nhat de truyen dat trang thai.
- Giu contrast va keyboard interaction co ban.

## Implementation
- Dung component/pattern co san.
- Giu state gan noi su dung neu khong can global.
- Khong them design system moi neu project da co convention.
