# Scope Rules

## Muc Tieu
Giu moi thay doi dung yeu cau va de review.

## Quy Tac
- Lam dung task user yeu cau truoc.
- Khong tranh thu refactor neu khong can.
- Khong sua file khong lien quan.
- Khong nang dependency neu task khong doi hoi.
- Khong thay doi public behavior neu user chi yeu cau cleanup.

## Khi Scope Tang
Bao ro neu can:
- Migration du lieu.
- Doi API contract.
- Doi auth/permission.
- Doi cau truc thu muc lon.
- Doi build/deploy pipeline.
