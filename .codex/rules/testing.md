# Testing Rules

## Chon Check
- Type changes: typecheck.
- Logic changes: unit/integration tests.
- UI changes: state/responsive/manual check.
- Build config: build.
- Security/config: secret/log/permission review.

## Chat Luong Test
- Test behavior, khong qua gan voi implementation.
- Uu tien test deterministic.
- Cover normal, edge, error case khi rui ro dang ke.
- Neu test fail do ly do khac, phan biet ro voi ket qua task.
