# Bugfix Debugging Skill

Use when something fails, crashes, behaves incorrectly, or regresses.

## Use With
- `roles/debugger.md`
- `workflows/bugfix.md`
- `rules/error-handling.md`
- `rules/testing.md`

## Procedure
1. Capture expected behavior, actual behavior, and reproduction clues.
2. Inspect logs, stack traces, tests, and recent related code.
3. Trace the smallest failing path.
4. Fix the root cause without broad rewrites.
5. Verify the failure no longer reproduces.

## Output
- Symptom.
- Root cause.
- Fix made.
- Verification result.
- Remaining risk.
