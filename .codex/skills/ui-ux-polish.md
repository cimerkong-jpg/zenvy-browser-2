# UI UX Polish Skill

Use for UI layout, interaction states, accessibility, responsive behavior, and visual cleanup.

## Use With
- `rules/frontend.md`
- `workflows/ui-polish.md`
- `checklists/ui.md`
- `roles/implementer.md`

## Procedure
1. Identify the primary user workflow.
2. Inspect existing component and style conventions.
3. Improve hierarchy, alignment, spacing, density, and feedback.
4. Cover loading, empty, disabled, focus, hover, error, and success states when relevant.
5. Verify responsive and accessibility basics when practical.

## Output
- UI changes made.
- States covered.
- Responsive/accessibility notes.
- Verification result.
