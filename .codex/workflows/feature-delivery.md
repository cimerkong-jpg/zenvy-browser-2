# Feature Delivery Workflow

## Steps
1. Xac dinh user goal va acceptance criteria.
2. Tim feature/pattern tuong tu.
3. Thiet ke thay doi nho nhat.
4. Implement.
5. Xu ly state/error/edge case.
6. Verify.
7. Bao cao ket qua.

## Done Means
- Main flow hoat dong.
- Edge case quan trong duoc xu ly.
- Check lien quan pass hoac rui ro duoc neu ro.
