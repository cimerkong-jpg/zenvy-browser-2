# Release Workflow

## Steps
1. Xem thay doi va muc tieu release.
2. Chay checks lien quan.
3. Kiem tra secret/debug/temp/machine-specific paths.
4. Kiem tra docs/changelog/setup neu can.
5. Neu blocker va readiness.

## Do Not
- Khong publish/tag/version bump neu user chua yeu cau.
- Khong noi release-ready neu core checks bi skip.
