# Review Workflow

## Steps
1. Xem diff hoac file duoc yeu cau.
2. Tim bug, regression, data loss, security issue truoc.
3. Kiem tra test gap.
4. Kiem tra maintainability va complexity.
5. Bao findings theo severity.

## Output Format
- Findings first.
- File/line references when possible.
- Open questions.
- Test gaps.
- Short summary.
